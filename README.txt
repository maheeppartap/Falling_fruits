****************************************Fruit Tetris 3d****************************************************
made by: Maheeppartap Singh	

This is the 3D version of the falling fruits.

To compile, simply write "make" in the terminal and then "./game3d" and the game will start.

Instructions:
	Move the arm around: 'w''a''s''d' keys
	move the blocks around: 'left arrow''right arrow'
	rotate the blocks: 'up arrow''down arrow'
	quit: 'q'
	restart:'r'
